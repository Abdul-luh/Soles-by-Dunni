# Soles by Dunni

Welcome to **Soles by Dunni**! This is a premium footwear showcase web application designed with modern web aesthetics in mind. It features a sleek dark mode interface, custom cursors, smooth scroll animations, and an elegant typography system.

## ✨ Features
- **Premium Design**: Built with a sophisticated color palette (`#0a0a0a` black, `#c8b89a` accent) for a luxurious feel.
- **Custom Cursor Interaction**: Enhances user engagement with dynamic cursor rings that respond to hover states.
- **Responsive Layout**: Designed to look stunning on both desktop and mobile devices.
- **Scroll Animations**: Elements fade and slide smoothly into view as you scroll down the page.
- **Dynamic Filtering**: Easily filter through the product catalog by category.

## 🛠 Technologies Used
- **HTML5**: Semantic structure and content.
- **CSS3**: Custom properties, CSS Grid, Flexbox, and advanced animations.
- **Vanilla JavaScript**: For IntersectionObserver (scroll animations), custom cursor logic, and dynamic product filtering.

## 🚀 Getting Started
Since this is a static website built with pure HTML, CSS, and JS, there is no complex build process required. 

To view this project locally, simply clone the repository and open the `index.html` file in your preferred web browser.

```bash
git clone <repository-url>
cd sole-by-dunni
# Open index.html in your browser
```
